Backbone Complety

Simple Backbone based solution for long and boring select boxes. It supports searching, multiple selection and custom templates for search results.

Basic Autocomplete

Autocomplete for Text Input

Autocomplete for Textarea

var complety = new Backbone.Complety({
  collection: gradeSix,
  targetContainer: '.input-complete',
  searchAttr: "name"
});
var completyArea = new Backbone.Complety({
  collection: gradeSix,
  targetContainer: '.area-complete',
  isArea: true,
  searchAttr: "name"
});
